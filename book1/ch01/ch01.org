* Getting Started

[[https://doc.rust-lang.org/book/ch01-00-getting-started.html][Link]]

** Basic info

*** Installation

Linux or macOS

#+begin_src

$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh

#+end_src

*** Checking installation

#+begin_src

$ rustc --version

#+end_src

*** Update

#+begin_src

$ rustup update

#+end_src

*** Uninstall

and

#+begin_src

$ rustup self uninstall

#+end_src

*** Local copy of documentation

#+begin_src

$ rustup doc

#+end_src

running the book offline

#+begin_src

$ rustup doc --book

#+end_src

*** Working offline with This Book

#+begin_src

$ cargo new get-dependencies
$ cd get-dependencies
$ cargo add rand@0.8.5 trpl@0.2.0

#+end_src

Afterwards no need to keep `get-dependencies/` folder.

Afterwards may use `--offline` flag with all `cargo` commands to use cached
versions instead of attempting to use the network
